# EcommercePOO

Lógica de Ecommerce criada com C# - Console.
Sistema de e-commerce completo aplicando 8 design patterns e os princípios SOLID em C#.
Builder → construção legível e segura do Pedido
Strategy → algoritmos de frete intercambiáveis
Decorator → descontos empilháveis sem modificar o Carrinho
Observer → notificações desacopladas de email, SMS e admin
Chain of Responsibility → validações sequenciais e extensíveis
State → máquina de estados com transições controladas
Factory Method → criação centralizada de processadores de pagamento
Facade → interface única que orquestra tudo
SOLID → presente em cada decisão de design ao longo do projeto
